# Java Practice Programs

This repository contains my collection of Java practice programs.  
Each program covers basic concepts of Java programming.

## 📂 Programs Included
- ArrayDemo.java – Working with arrays
- Calculator.java – Simple calculator operations
- Factorial.java – Factorial calculation
- Fibbonacci.java – Fibonacci series
- HelloWorld.java – First Java program
- PrimeCheck.java – Prime number check
- SimpleInterest.java – Interest calculation
- Sorting.java – Sorting algorithms
- StudentMarks.java – Student marks calculation
